Apa Itu Method? 🍼

Bayangin kamu punya robot, dan kamu bilang ke dia: "Robot, bersihin kamar!" Nah, perintah "bersihin kamar" itu adalah method. Jadi, method = perintah yang robot bisa lakuin.

Contoh Method di Java 👨‍💻

public class Robot {
  void bersihKamar() {
    System.out.println("Kamar sudah bersih!");
  }
}

Apa Itu Function? 🍭

Function kayak method juga, tapi dia bisa ngasih kamu hadiah alias *hasil*. Misal kamu tanya: "Function, berapa 2+2?" dia jawab: "4"

Contoh Function di Java 📐

int tambah(int a, int b) {
  return a + b;
}

Apa Itu Procedure? 🛠️

Procedure itu kayak method juga, tapi dia biasanya gak ngasih hasil. Kayak disuruh nyapu halaman. Dia kerja, tapi gak bilang "saya udah selesai", dia tinggal pergi aja wkwk.

Contoh Procedure di Java 🧹

void nyapuHalaman() {
  System.out.println("Halaman disapu bersih");
}

Apa Itu Parameter? 📦

Parameter itu kayak barang bawaan buat method atau function. Misalnya kamu bilang: "Robot, nyalain lampu di kamar!" Nah, "di kamar" itu parameternya.

Contoh Parameter di Java 🔌

void nyalainLampu(String ruangan) {
  System.out.println("Lampu di " + ruangan + " dinyalakan.");
}
